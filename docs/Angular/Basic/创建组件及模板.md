# 创建组件及模板

## 一、创建组件
* 创建组件
`ng g component components/header`
* 使用组件
`<app-home></app-home>`
## 二、数据绑定
1. angular中使用{{}}绑定业务逻辑里面定义的数据
```html
  <h2>{{title}}</h2>
```
2. angular模板里面允许做简单的计算
```html
  <p>1+2={{1+2}}</p>
```
3. 模板里面绑定html
```ts
public content:any = '<h2>我是一个html标签，用[innerHtml]解析</h2>';
```
```html
  <p [innerHtml]='content'></p>
```
